Claude Code Lab 5분 오리엔테이션

이 사이트는 Claude Code 사용법만 배우는 곳이 아니라, 디자이너가 Claude와 함께 하나의 프로덕트를 기획부터 배포까지 완주하는 실습 코스입니다.

예상 시간 5분 난이도 입문 결과물 학습 방향 이해
왜 필요한가 완주 흐름 역할 구분 어디서 시작할까

왜 디자이너에게 Claude Code가 필요한가

디자이너의 일은 화면을 그리는 데서 끝나지 않습니다. 문제를 정의하고, 흐름을 설계하고, 구현 가능성을 확인하고, 배포된 결과물을 검수하는 일까지 점점 더 가까워지고 있습니다. Claude Code는 이 과정을 혼자 전부 외우게 하는 도구가 아니라, 각 단계를 같이 밀고 가는 작업 파트너입니다.

이 사이트의 완주 흐름

진단 기획 설계 디자인 개발 검수 배포 자동화

사이트, Claude Code, 브라우저의 역할

01

사이트

단계별 설명, 프롬프트, 체크리스트를 제공합니다.

02

Claude Code

파일을 읽고, 수정하고, 실행 가능한 화면을 만듭니다.

03

브라우저

결과 화면을 보고 디자이너 기준으로 검수합니다.

04

디자이너

문제, 의도, 품질 기준을 정하고 반복 개선을 지휘합니다.

어디서 시작하면 좋을까

Claude Code가 처음이면 활용 가이드를 먼저 보고, 이미 대략 알고 있다면 내 수준 진단으로 시작 지점을 정하세요.
