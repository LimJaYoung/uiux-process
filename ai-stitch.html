언제 쓰나

와이어프레임 전에 여러 UI 방향을 빠르게 보고 싶거나, 모바일과 웹 화면을 동시에 탐색하고 싶을 때 씁니다.

활용 방법

  • 화면 목적, 사용자, 핵심 컴포넌트를 명확히 적습니다.
  • 여러 시안을 만든 뒤 정보 위계와 카드 리듬을 비교합니다.
  • 좋은 시안을 Figma나 Codex 작업 기준으로 옮깁니다.
  • 생성 결과를 그대로 쓰기보다 디자인 규칙으로 추출합니다.

바로 쓰는 프롬프트

Create a responsive landing page for a UIUX designer portfolio course.

Requirements:
- Large editorial hero
- Black, white, and gray palette
- Cards for AI tools and portfolio outputs
- Clear CTA for diagnosis
- Mobile-first layout
- Premium product design feeling

Avoid:
- Generic startup gradient
- Too many colors
- Decorative shapes without function
