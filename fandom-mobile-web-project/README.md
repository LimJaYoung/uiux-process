# AURORA:NE Fan Orbit

Advanced 팬덤 모바일웹 제작 증명 프로젝트입니다.

주제는 가상의 4인조 퍼포먼스 그룹 `AURORA:NE`의 팬덤 모바일웹이며, 430px 이하 모바일 화면을 기준으로 설계했습니다. 목표는 단순 랜딩페이지가 아니라 팬덤앱의 핵심 기능을 갖춘 포트폴리오 결과물입니다.

## Stack

- React
- TypeScript
- Tailwind CSS 기준의 utility 설계
- 배포 검증용 정적 결과물: `dist/index.html`

## 포함 기능

- Home: 팬덤 세계관, 오늘의 라이브, 인기 피드, 진행 중 투표
- Members: 멤버 프로필 카드
- Feed: 팬 피드와 응원 반응
- Vote: 응원 투표 진행률과 참여 상태
- Live: 라이브 일정과 알림 신청
- Chat: 채팅 목록, 채팅방, 말풍선, 입력창, 전송/읽음 상태, 안전 UX
- My: 활동 배지, 알림, 참여 기록

## 산출물

- `docs/process.md`: 1번 아이템 선정부터 17번 케이스스터디까지 전체 진행 기록
- `docs/quality-gate.md`: 최상급 품질 기준과 QA 결과
- `docs/deploy-info.md`: 배포 준비 정보
- `src/`: React/TypeScript 소스
- `dist/`: 바로 열 수 있는 검증용 결과물
