Claude는 HMW, 페르소나, 사용자 여정처럼 긴 흐름을 정리하기 좋고, ChatGPT는 케이스 스터디 문장을 여러 톤으로 다듬기 좋습니다.

문제 정의 프롬프트 UX 설계 프롬프트 Claude ChatGPT

Codex로 HTML, CSS, JS 파일을 직접 수정하고, Claude 계열 도구로 코드 구조 설명과 수정 방향을 보완해 배포 가능한 화면으로 만듭니다.

구현 프롬프트 배포 프롬프트 Codex Claude

근거 수집과 반복 정리

근거 수집 프롬프트 Perplexity · n8n ChatGPT
07 Visual Board · Midjourney · Higgsfield · Figma AI

Midjourney로 이미지 방향을 만들고, Higgsfield로 움직임을 더하고, Figma에서 실제 화면 톤과 맞는지 검토합니다.

Process by Level

레벨에 맞는 프로세스

Beginner는 첫 MVP를 작게 완주하고, Intermediate는 SKILL·PRD·Design.md로 다중 화면과 디자인 시스템까지 구조화합니다.

Beginner · Simple MVP

첫 MVP를 프롬프트 흐름대로 완주하기

아이템 선정, 리서치, 문제 정의, UX 설계, 구현, 배포, 케이스 스터디를 한 페이지의 복사형 프롬프트로 따라갑니다.

  3. Codex로 구현, QA, Vercel 배포 URL까지 연결
Intermediate · Harness production

AI 에이전트팀으로 서비스형 포트폴리오 만들기

SKILL, 에이전트팀, PRD, Design.md, QA 리포트를 연결해 다중 화면과 디자인 시스템이 있는 포트폴리오를 만듭니다.

SKILL 4 PRD Design.md
Intermediate 전체 가이드 보기
  1. SKILL 4종으로 기획자, 디자이너, 개발자, QA 역할 설정

  3. Design.md와 tokens.css로 다중 화면 품질 관리
Advanced · Harness + n8n

자동으로 업데이트되는 운영형 포트폴리오 만들기

병렬 에이전트, 코디네이터 SKILL, n8n, GitHub Actions, 코드 에이전트를 연결해 운영 시스템을 설계합니다.

Parallel AI n8n Auto deploy
Advanced 전체 가이드 보기
  1. 병렬 에이전트와 코디네이터로 작업을 분산하고 취합
  2. n8n으로 콘텐츠 수집, 영상 연동, 폼 처리를 자동화
  3. GitHub Actions와 Vercel로 자동 배포 파이프라인 구축

Daily Curation

AI + 디자인 뉴스와 유튜브 큐레이션

가능합니다. GitHub Actions가 매일 JSON을 갱신하고, Vercel이 자동 배포하면 정적 사이트도 매일 업데이트됩니다.

AI + Design News

매일 수집되는 AI 제품, UX, 디자인 시스템 뉴스

RSS와 공식 블로그를 수집해 제목, 출처, 링크, 짧은 요약만 안전하게 노출합니다.

YouTube Curation

디자이너를 위한 영상 학습 큐레이션

채널 RSS 또는 YouTube API를 통해 최신 영상 목록을 JSON으로 업데이트합니다.

Automation MVP

자동 수집 → JSON 업데이트 → Vercel 배포

API 키는 프론트에 넣지 않고 GitHub Secrets에 보관하고, 저작권 문제를 피하기 위해 링크와 짧은 요약만 보여줍니다.

FAQ

자주 묻는 질문

5분 오리엔테이션
코드를 전혀 몰라도 시작할 수 있나요?

처음에는 HTML, CSS, JavaScript 파일을 열고 AI 도구와 함께 수정하는 방식으로 시작합니다.

Figma 디자인을 바로 개발로 바꾸는 사이트인가요?

그보다 앞단의 기획, 설계, 디자인 기준까지 함께 정리해서 구현 가능한 요청으로 바꾸는 흐름입니다.

최종 결과물은 무엇인가요?

브라우저에서 열리는 프로토타입, 배포 URL, 포트폴리오에 설명할 수 있는 제작 기록입니다.

자료실은 언제 쓰나요?

프롬프트 템플릿, AI 워크플로우 가이드, QA 체크리스트가 필요할 때 바로 열어 씁니다.
